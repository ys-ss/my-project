FLOCSS

FLOCSSとは

「Foundation Layout Object CSS」の略。
OOCSSやBEM、SMACSSのいいとこ取りをしたCSS設計思想。
FLOCSSは次の3つのレイヤーと、Objectレイヤーの子レイヤーで構成されている。

Foundation

Reset.cssやNormalize.cssなどサイト全体のデフォルトスタイルを管理する。

Layout

各ページを構成するヘッダーやフッターなどのサイト全体で共通したエリアを管理する。

Object

サイト全体で再利用できるパターンを持つモジュールを管理する。
FLOCSSでのObjectは、さらに次の3つのレイヤーに分けられている。

Component
再利用できるパターンとして、小さな単位のモジュールを定義する。
Project
いくつかのComponentと、それに該当しない要素によって構成される大きな単位のモジュールを定義する。
Utility
ComponentとProjectのモディファイアで解決することが難しい・適切では無い、わずかなスタイルの調整のための便利クラスなどを定義する。

MindBEMding

BEMシステムのシンタックスである、Block、Element、Modifierに分類して構成される規則を採用されている。

JavaScriptで操作され、状態を変化させるためのクラス

JavaScriptで操作されるような「状態」を表すようなModifierについては、SMACSSのStateパターンの命名を拝借し、is-*プレフィックスを付与し、.is-activeというようにする。

プレフィックス

役割を明確にするためにプレフィックスをつけることを推奨する。

  • Layout - .l-*
  • Component - .c-*
  • Project - .p-*
  • Utility - .u-*
