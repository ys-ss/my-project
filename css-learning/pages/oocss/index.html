OOCSS

OOCSSとは

OOCSSは、Object Oritented CSS（オブジェクト指向CSS）の略。
OOCSSでは、以下の2点がポイントとなる。

  • ストラクチャー（構造）とスキン（見た目）の分離
  • コンテナとコンテンツの分離

ストラクチャーとスキンの分離

「ストラクチャー（横幅や高さなど）とスキン（文字色や背景色など）」の組み合わせで成り立っていると考えるのが「ストラクチャーとスキンの分離」の考え方となる。

HTML

<div id="contents">
  <button class="btn general">通常ボタン</button>
  <button class="btn warning">警告ボタン</button>
<div>

CSS

/* ストラクチャー */
.btn {
  display: inline-block;
  width: 100px;
  padding: 10px 0;
  text-align: center;
  border: none;
}

/* スキン */
#contents .general {
  color: #fff;
  background-color: #666;
}

#contents .warning {
  color: #fff;
  background-color: #e4CC2D;
}

コンテナとコンテンツの分離

「コンテナとコンテンツの分離」とは「モジュールをなるべく特定のエリアに依存させない」という方針を指す。

部品（コンテンツ）が、他の場所でも再利用ができるように親要素（コンテナ）に依存しないようにする。

HTML

<div id="contents">
  <button class="btn">通常ボタン</button>
<div>

CSS

/* × 親要素に依存している */
#contents .btn {
  display: inline-block;
  width: 100px;
  padding: 10px 0;
  text-align: center;
  border: none;
}

/* ○ 親要素に依存していないため、再利用ができる */
.btn {
  display: inline-block;
  width: 100px;
  padding: 10px 0;
  text-align: center;
  border: none;
}
