CSS設計とは

CSSを設計する時のルール。
開発するWebサイトの性質や規模によって、最適なCSS設計を用いることで、「よいCSS」にすることができる。

よいCSS設計が目指す4つのゴール

「よいCSSとは何か」を考える際に役立つ指針が「よいCSS設計が目指す4つのゴール」である。

予測できる
新しいスタイリングを追加する、または既存のスタイリングを更新しても、自分の意図しない箇所に影響を与えないよう設計する。
再利用できる
既存のパーツを別の箇所でも使用したいとなったとき、コードをいちいち書き直したり上書きする手間がない状態にする。
保守できる
新しいモジュールや機能を追加・更新、あるいは配置換えしたとき、既存のCSSをリファクタリングする必要がない状態にする。
拡張できる
CSSに携わる人が一人であっても、複数からなるチームであっても、問題なく管理できる状態にする。
